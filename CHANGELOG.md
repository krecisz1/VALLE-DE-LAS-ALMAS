## [1.0.0] - 2024-01-22

### ✨ Agregado

#### 🌟 Características Principales
- **Sistema de memoriales digital** completo para honrar la memoria de seres queridos
- **Modelo freemium** con plan gratuito y plan premium "Tributo Completo"
- **Soporte para personas y mascotas** en los memoriales
- **Interfaz responsive** optimizada para móvil, tablet y desktop

#### 👑 Plan Gratuito
- Creación de **1 memorial básico**
- **Información personal completa** (nombre, fechas, biografía, ubicación, ocupación)
- **Foto principal** de perfil con sistema de selección intuitivo
- **Biografía ilimitada** para contar historias de vida
- **Visualización pública** para compartir recuerdos
- **Comentarios básicos** (hasta 5 por memorial)

#### 🎯 Plan Tributo Completo ($4.99/mes)
- **Memoriales ilimitados** para toda la familia
- **Galería de fotos completa** con títulos personalizados
- **Videos y multimedia** para preservar momentos especiales
- **Árbol genealógico interactivo** con 23 tipos de relaciones familiares
- **Códigos QR para lápidas** para acceso rápido desde cementerios
- **Comentarios ilimitados** de familiares y amigos
- **Búsqueda avanzada** y filtros
- **Configuración de privacidad** granular
- **Soporte prioritario**

#### 🔗 Árbol Genealógico
- **23 tipos de relaciones familiares** organizadas por categorías:
  - **👨‍👩‍👧‍👦 Familia Directa**: Padre, Madre, Hijo, Hija, Hermano, Hermana
  - **👴👵 Abuelos y Generaciones**: Abuelo, Abuela, Nieto, Nieta, Bisabuelo, Bisabuela, Bisnieto, Bisnieta
  - **👨‍👦 Familia Extendida**: Tío, Tía, Sobrino, Sobrina, Primo, Prima
  - **💑 Familia Política**: Suegro, Suegra, Yerno, Nuera, Cuñado, Cuñada
  - **💕 Otros**: Cónyuge, Pareja, Mascota
- **Visualización interactiva** del árbol familiar
- **Navegación bidireccional** entre relaciones
- **Sistema de relaciones inversas** automático

#### 🎨 Interfaz de Usuario
- **Design system consistente** con Tailwind CSS v4
- **Componentes UI modernos** usando shadcn/ui y Radix UI
- **Iconografía cohesiva** con Lucide React
- **Animaciones suaves** para una mejor experiencia
- **Modo claro y oscuro** automático
- **Accesibilidad** implementada en todos los componentes

#### 📱 Funcionalidades de Acceso
- **Sistema de login** con validación de email
- **Acceso QR** para visitantes sin cuenta
- **Permisos granulares** por memorial
- **Modo demo** con memoriales de ejemplo
